Step 1
Real-world problem

Example: exam timetabling.
Courses sharing students must not be scheduled at the same time.

Step 2
Reduce to graph coloring

Vertices = courses
Edges = “cannot share a time slot” constraints
Colors = time slots

Step 3
Build an oracle

Create a circuit that checks constraints.
Mark valid assignments (phase flip).

Step 4
Solve with Grover

Run Grover iterations, then measure.
The result concentrates on a valid coloring.
